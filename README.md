# Meu Site Pessoal

Bem-vindo ao repositório do meu site pessoal! Este projeto é um portfólio simples e interativo, desenvolvido para exibir minhas habilidades e projetos.

## Funcionalidades

-   **Design Responsivo:** O site se adapta a diferentes tamanhos de tela.
-   **Animações Interativas:** Elementos visuais dinâmicos para uma experiência de usuário envolvente.
-   **Cursor Personalizado:** Um cursor único para melhorar a interatividade.
-   **Status do Discord:** Exibe meu status atual no Discord (se configurado).

## Tecnologias Utilizadas

-   HTML5
-   CSS3
-   JavaScript

## Como Rodar Localmente

Siga estas instruções para configurar e rodar o projeto em sua máquina local.

### Pré-requisitos

Você precisará de um navegador web moderno para visualizar o site.

### Instalação

1.  Clone o repositório:
    ```bash
    git clone https://github.com/rreis-nt/meu-site.git
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd meu-site
    ```

### Executando o Site

Basta abrir o arquivo `index.html` no seu navegador preferido.

```bash
# Exemplo no Windows
start index.html

# Exemplo no macOS/Linux
open index.html
```
